## Filing Process

### Step 1: Pre-Filing Search (Recommended)
Check for conflicts:
```
1. Visit: https://tmsearch.uspto.gov
2. Search for:
   - "TWZRD"
   - "WIZARD" (phonetic equivalent)
   - Similar marks in Classes 9, 42
3. Look for:
   - Identical marks
   - Similar marks in related industries
   - Dead/abandoned marks (you can still use)
```

**Expected Result:** Should be clear as of 2025-11-11

### Step 2: Gather Materials
- [ ] Specimens ready (screenshots saved)
- [ ] Dates of first use confirmed
- [ ] Business entity details confirmed
- [ ] Payment method ready ($250 per class × number of classes)

### Step 3: File Application
**Option A: DIY via TEAS Plus**
```
1. Visit: https://www.uspto.gov/trademarks/apply/initial-application-forms
2. Select "TEAS Plus" ($250/class)
3. Create USPTO.gov account
4. Fill out form:
   - Mark: TWZRD
   - Type: Standard Character Mark
   - Owner: TWZRD Inc.
   - Goods/Services: Copy descriptions above
   - Upload specimens
   - Enter dates of first use
5. Review (CAREFULLY - corrections cost $100+)
6. Pay ($250 × number of classes)
7. Submit
8. Save confirmation page
```

**Option B: Hire Trademark Attorney**
- Cost: $500-$1500 + filing fees
- Handles search, filing, office actions
- Recommended if budget allows
- Find at: USPTO Trademark Attorney Directory

### Step 4: Monitor Application
After filing:
- You'll receive Serial Number immediately
- Check status: https://tsdr.uspto.gov
- Expect examiner review in 3-4 months
- Respond to any Office Actions within 6 months

### Step 5: Publication & Registration
Timeline:
- **~3 months:** Examiner review
- **If approved:** Published in Official Gazette (30-day opposition period)
- **If no opposition:** Registration certificate issued
- **Total time:** 8-12 months from filing to registration

## Common Office Action Issues

Be prepared to respond if examiner asks:

### Likelihood of Confusion
- Similar existing mark found
- Response: Argue differences (goods, channels, meaning)

### Merely Descriptive
- Mark describes goods/services too directly
- Response: Argue distinctiveness, acquired meaning

### Specimen Issues
- Specimen doesn't show use in commerce
- Response: Submit better specimen with explanation

### Unacceptable ID (Identification)
- Goods/services description too vague
- Response: Use examiner's suggested wording

**Cost to respond:** Usually free if DIY, $500-$1000 if attorney

### Maintenance Filings (Critical - Missing = Cancellation!)

**Section 8 Declaration (Between years 5-6):**
- File: Years 5-6 after registration
- Fee: $225-$525
- Must prove continued use in commerce

**Section 9 Renewal (Every 10 years):**

## Important Notes

1. **Use the ™ symbol NOW** (before registration)
   - You can use ™ immediately since you're using it in commerce
   - Switch to ® only AFTER registration is complete

2. **Consistent Use**
   - Always use "TWZRD" or "TWZRD™" (consistent styling)
   - Never use as verb ("twzrding") or plural
   - Always capitalize: TWZRD (not Twzrd or twzrd)

3. **Protect Your Mark**
   - Monitor for infringement
   - Send cease & desist if needed
   - Keep evidence of your use

4. **International Protection**
   - USPTO only covers USA
   - For global: File "Madrid Protocol" application
   - Cost: ~$1000-$3000 for multiple countries
