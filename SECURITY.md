# Security Policy

## Reporting Security Vulnerabilities

We take security seriously. If you discover a security vulnerability, please follow responsible disclosure practices.

### How to Report

**Email**: security@twzrd.xyz

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes

### Response Time

We will acknowledge receipt within 48 hours and provide a detailed response within 7 days.

### Scope

The following are in scope for security reports:
- On-chain program vulnerabilities
- Economic exploits
- Access control bypasses
- Data validation issues

### Out of Scope

- Issues in dependencies without direct impact
- Best practice suggestions without security impact
- Issues requiring physical access

### Recognition

We appreciate security researchers who follow responsible disclosure. With your permission, we will acknowledge your contribution in our security updates.

Thank you for helping keep Attention Oracle secure.
