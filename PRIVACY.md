# Privacy Notice (On-Chain Programs)

This repository contains only on-chain Solana programs. It does not operate user-facing services or collect off-chain user data.

## On-chain data

This program runs on the public Solana blockchain. All data submitted to these programs is public by default.

The programs store and emit only on-chain state:

- Public keys (wallet addresses)
- Protocol state (program configuration, channel state, claim state)
- Transaction metadata emitted by Solana

## Not stored by these programs

- Email addresses, names, or other personal identifiers
- IP addresses or device identifiers
- Off-chain browsing or usage data

For questions, open an issue in this repository.
