# Twzrd Attention Oracle Privacy Policy

Effective: December 15, 2025

Twzrd is an open-source Solana program and claims portal (https://github.com/twzrd-sol/attention-oracle-program) that distributes CCM tokens based on on-chain attention signals.

## Data Collected

- Public creator identifier (via third-party OAuth readonly scope)
- Solana wallet address (via signature verification)
- No email, name, or private profile data collected

## Use

- Identifier linked to wallet for reward eligibility
- Data stored in managed PostgreSQL for oracle computation only

## Sharing

- No data sold or shared with third parties

## Security

- Private keys never leave user device
- All code public under MIT/Apache 2.0 licenses
