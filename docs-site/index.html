What is TWZRD?

TWZRD is an open-source attention oracle on Solana that tokenizes viewer presence in live streams. It provides verifiable, on-chain proof of attention and distributes tokens based on verified engagement.

Quick Start

TypeScript SDK

npm install @twzrd/sdk

import { TwzrdClient } from '@twzrd/sdk';
import { Connection } from '@solana/web3.js';

const client = new TwzrdClient(
  new Connection('https://api.mainnet-beta.solana.com')
);

const channelState = await client.getChannelState(streamerPubkey);

Rust SDK

cargo add twzrd-sdk

use twzrd_sdk::TwzrdClient;

let client = TwzrdClient::new("https://api.mainnet-beta.solana.com");
let channel_state = client.get_channel_state(&streamer_pubkey).await?;

CLI

cargo install twzrd-cli

twzrd info
twzrd channel <STREAMER_PUBKEY>
twzrd claim <USER_PUBKEY> <CHANNEL_PUBKEY>

Architecture

TWZRD consists of three main components:

  1. Solana Program - On-chain logic for channel management and token claims
  2. Web Frontend - User interface for claiming tokens via Twitch OAuth
  3. SDKs - TypeScript and Rust libraries for programmatic access
