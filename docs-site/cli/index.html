Installation

cargo install twzrd-cli

Or build from source:

git clone https://github.com/twzrd-sol/attention-oracle-program
cd attention-oracle-program/rust-packages/twzrd-cli
cargo install --path .

Global Options

--rpc-url <URL>     Solana RPC endpoint
                      Default: https://api.mainnet-beta.solana.com
--help              Show help message
--version           Show version

Commands

info

Display TWZRD program information.

twzrd info

Output:

TWZRD Attention Oracle
Program ID: [PROGRAM_ID]
Version: 0.1.0
Website: https://twzrd.xyz

channel

Get channel state for a streamer.

twzrd channel <STREAMER_PUBKEY>

Example:

twzrd channel 7xKXtg2CW87d97TXJSDpbD5jBkheTqA83TZRuJosgAsU

Output:

Channel State:
  Streamer: 7xKXtg2CW87d97TXJSDpbD5jBkheTqA83TZRuJosgAsU
  Mint: [MINT_ADDRESS]
  Current Epoch: 12345
  Total Minted: 1500000
  Active: true

claim

Claim tokens for a user.

twzrd claim <USER_PUBKEY> <CHANNEL_PUBKEY>

Example:

twzrd claim 9xQeWvG816bUx9EPjHmaT23yvVM2ZWbrrpZb9PusVFin 7xKXtg2CW87d97TXJSDpbD5jBkheTqA83TZRuJosgAsU

Output:

Claiming tokens...
Transaction signature: [SIGNATURE]
Success! Tokens claimed.

Examples

Check Channel on Devnet

twzrd --rpc-url https://api.devnet.solana.com channel [PUBKEY]

Claim with Custom RPC

twzrd --rpc-url https://your-rpc.com claim [USER] [CHANNEL]

Environment Variables

SOLANA_RPC_URL      Override default RPC URL
RUST_LOG            Set logging level (debug, info, warn, error)
