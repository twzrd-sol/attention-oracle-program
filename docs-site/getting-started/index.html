Basic Usage

Connect to TWZRD

import { TwzrdClient } from '@twzrd/sdk';
import { Connection, PublicKey } from '@solana/web3.js';

// Connect to Solana mainnet
const connection = new Connection('https://api.mainnet-beta.solana.com');
const client = new TwzrdClient(connection);

// Get channel information
const streamerPubkey = new PublicKey('YOUR_STREAMER_PUBKEY');
const channelState = await client.getChannelState(streamerPubkey);

console.log('Channel epoch:', channelState.currentEpoch);
console.log('Total minted:', channelState.totalMinted);

Core Concepts

Channels

Each streamer has a channel represented on-chain. Channels manage their own mint and distribute tokens to verified viewers.

Epochs

Time is divided into 5-minute epochs. Viewers can claim tokens once per epoch if they're present during that time window.

Claims

Viewers authenticate via Twitch OAuth and submit claims to receive tokens. The program verifies presence and mints tokens to the user's Solana wallet.
