Streamer Integration Guide

1. Prerequisites

  • Twitch account
  • Solana wallet (Phantom, Solflare, etc.)
  • Small amount of SOL for transactions

2. Initialize Your Channel

Contact TWZRD team to initialize your channel:

Email: dev@twzrd.xyz
Subject: Channel Initialization Request
Include: Twitch username, Solana wallet address

3. Share Claim Link

Once initialized, share your claim link with viewers:

https://twzrd.xyz/claim.html?channel=[YOUR_CHANNEL_ID]

4. Monitor Your Channel

twzrd channel [YOUR_PUBKEY]

Developer Integration Guide

1. Install SDK

npm install @twzrd/sdk @solana/web3.js

2. Initialize Client

import { TwzrdClient } from '@twzrd/sdk';
import { Connection } from '@solana/web3.js';

const connection = new Connection(
  process.env.SOLANA_RPC_URL || 'https://api.mainnet-beta.solana.com'
);
const client = new TwzrdClient(connection);

3. Fetch Channel Data

const channelState = await client.getChannelState(streamerPubkey);
console.log('Current epoch:', channelState.currentEpoch);
console.log('Total minted:', channelState.totalMinted.toString());

4. Handle Claims

try {
  const signature = await client.claimTokens(userPubkey, channelPubkey);
  console.log('Claim successful:', signature);
} catch (error) {
  console.error('Claim failed:', error.message);
}

Wallet Setup Guide

1. Install a Solana Wallet

  • Phantom (Recommended)
  • Solflare
  • Backpack

2. Get Some SOL

You'll need a small amount of SOL for transaction fees. Purchase from:

  • Coinbase
  • Binance
  • Jupiter (swap from other tokens)

3. Connect to TWZRD

  1. Visit twzrd.xyz/claim.html
  2. Click "Connect Wallet"
  3. Approve connection in your wallet
  4. Authenticate with Twitch
  5. Claim your tokens!

Advanced Usage

Batch Processing

// Process multiple channels
const channels = [pubkey1, pubkey2, pubkey3];
const states = await Promise.all(
  channels.map(ch => client.getChannelState(ch))
);

states.forEach((state, i) => {
  console.log(`Channel ${i}: Epoch ${state.currentEpoch}`);
});

Custom Program Interaction

import { TwzrdClient, PROGRAM_ID } from '@twzrd/sdk';
import { PublicKey, TransactionInstruction } from '@solana/web3.js';

// Derive channel PDA
const [channelPDA] = PublicKey.findProgramAddressSync(
  [Buffer.from('channel'), streamerPubkey.toBuffer()],
  PROGRAM_ID
);

Monitoring Events

// Subscribe to account changes
connection.onAccountChange(
  channelPDA,
  (accountInfo) => {
    console.log('Channel state updated:', accountInfo);
  },
  'confirmed'
);
